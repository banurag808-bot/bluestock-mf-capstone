# 📊 Bluestock Mutual Fund Analytics & Investor Insights Platform

## 📖 Project Overview

The **Bluestock Mutual Fund Analytics & Investor Insights Platform** is an end-to-end Data Analytics Capstone Project focused on analyzing the Indian mutual fund industry. The project combines data engineering, exploratory data analysis (EDA), financial performance analytics, advanced risk analysis, and interactive business intelligence dashboards to generate actionable insights for investors and fund managers.

---

# 🎯 Business Objectives

* Analyze mutual fund industry trends.
* Evaluate fund performance using financial metrics.
* Measure risk-adjusted returns.
* Study investor behaviour and transaction patterns.
* Analyze SIP growth trends.
* Build an interactive Power BI dashboard.
* Generate business recommendations for investors and asset management companies.
